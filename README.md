```sh
./install.sh
```
### Run locally
[run.sh](https://github.com/AlgoSolver/client/blob/main/scripts/run.sh)
```sh
./run.sh
```

[run_docker.sh](https://github.com/AlgoSolver/client/blob/main/scripts/run_docker.sh)
```sh
./run_docker.sh
```
